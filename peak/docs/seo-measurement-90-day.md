# SEO Measurement Plan (90 Days)

## Goal

Grow organic visibility and reservations from local search queries:

- `internet cafe suceava`
- `playstation suceava`
- `pret internet cafe suceava`
- `rezervare internet cafe suceava`

## Setup (Week 1)

1. Google Search Console
   - Verify domain property.
   - Submit sitemap: `https://peak-gaming.site/sitemap.xml`.
   - Inspect and request indexing for:
     - `/`
     - `/contact`
     - `/internet-cafe-suceava`
     - `/playstation-suceava`
2. Analytics/events
   - Track reservation starts and completed reservations.
   - Track click-to-call events on `/contact`.
3. Rank tracking baseline
   - Record current position for primary keywords weekly.

## KPI Dashboard

- Impressions (non-brand local queries)
- Average position (keyword cluster)
- CTR by landing page
- Organic users to:
  - `/internet-cafe-suceava`
  - `/playstation-suceava`
  - `/rezervare`
- Reservation conversion rate from organic traffic

## Weekly Cadence

- Review top queries and pages in Search Console.
- Improve title/meta on pages where:
  - impressions are rising
  - CTR is low
- Add 1-2 FAQ entries on best-performing local pages.

## Monthly Cadence

- Refresh one landing page section (new offer, new FAQ, clearer CTA).
- Add one local proof signal:
  - event mention
  - new testimonials
  - updated photos
- Audit internal links from home/prices/reservation/contact.

## Success Targets by Day 90

- Local landing pages indexed and stable in search.
- Strong growth in impressions for Suceava-intent keywords.
- CTR increase on core pages after metadata iterations.
- Measurable uplift in reservation actions from organic traffic.
